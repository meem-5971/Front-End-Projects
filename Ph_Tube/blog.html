Variable declared with var is a global variable. If we declare a variable in a function using var, we can access this variable throughout this function. On the other hand, both let and const are block-scoped. That means we cant not use the variables outside their block boundary. But the main difference between let and const is, we cant change the value of a variable declared with const. But let has this opportunity. Its dynamic.

Null is intentionally assigned. A developer can assign Null to a variable to represent that there is no value or this variable is empty.
But, undefined is automatically assigned. Or, we can say, the variable is declared but not assigned any value.

REST API is an architectural style of designing applications. It is approached to build connections between computer systems on the internet. A REST API uses commands to obtain resources. With REST, networked components are a resource the user requests access to, whose implementation details are unclear. It provides a flexible and scalable way to build web services.
